1. API 키 발급 (무료)

  1. Google AI Studio 접속 후 구글 로그인
  2. "API 키 만들기" 버튼 클릭
  3. 생성된 키를 복사하여 위 입력란에 붙여넣기

2. 유료 플랜 전환 (선택)

무료 플랜으로 블로그, 쓰레드 포스팅은 제약 없이 충분히 사용 가능합니다.
단, 쓰레드 이미지 생성은 무료 플랜의 요청 제한으로 인해 유료 플랜을 권장합니다.

  1. Google AI Studio 접속
  2. 좌측 메뉴 "설정" → "결제" 클릭
  3. 유료 플랜 활성화 후 결제 정보 입력

API 키는 브라우저에만 저장되며 서버로 전송되지 않습니다. (요청 시에만 사용)
